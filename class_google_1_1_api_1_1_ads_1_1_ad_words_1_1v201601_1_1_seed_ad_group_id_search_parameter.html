Detailed Description

A SearchParameter for

KEYWORD IdeaTypes that specifies that the supplied AdGroup should be used as a seed for generating new ideas. For example, an AdGroup's keywords would be used to generate new and related keywords.

This element is supported by following IdeaTypes: KEYWORD.

This element is supported by following RequestTypes: IDEAS.
