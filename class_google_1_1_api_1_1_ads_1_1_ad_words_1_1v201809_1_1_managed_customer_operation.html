Detailed Description

An operation to create or modify a managed customer.

Note: The REMOVE operator is not supported.

Property Documentation

string Google.Api.Ads.AdWords.v201809.ManagedCustomerOperation.inviteeEmail
getset

Email invitee.

Email address of the user who should be invited on the created managed customer. Accessible to whitelisted customers only.

Note: This is only supported for ADD operator. This field is read only and will be ignored when sent to the API for the following Operators: REMOVE and SET.

AccessRole Google.Api.Ads.AdWords.v201809.ManagedCustomerOperation.inviteeRole
getset

Email invitee role.

The proposed role ofcode inviteeEmail} on the created managed customer. Accessible to whitelisted customers only.

Note: This is only supported for ADD operator. This field is read only and will be ignored when sent to the API for the following Operators: REMOVE and SET.
