PolicyTopicEvidenceDestinationMismatchUrlType [] Google.Api.Ads.AdWords.v201809.PolicyTopicEvidence.policyTopicEvidenceDestinationMismatchUrlTypes
getset

The url types that triggered this policy topic to be reported. This field is associated with the policyTopicEvidenceType and will only be populated when it has value of DESTINATION_MISMATCH. So for example, when policyTopicEvidenceType is DESTINATION_MISMATCH, policyTopicEvidenceDestinationMismatchUrlTypes will be populated with mismatched url types, DISPLAY_URL, FINAL_URL and/or FINAL_MOBILE_URL.
