Detailed Description

Logical expression for targeting webpages of an advertiser's website.

A condition is defined as operand OP argument where operand is one of the values enumerated in WebpageConditionOperand, and, based on this value, OP is either of EQUALS or .

  • EQUALS is only allowed for WebpageConditionOperand::CATEGORY, WebpageConditionOperand::CUSTOM_LABEL, and WebpageConditionOperand::URL operands.
  • CONTAINS is only allowed for WebpageConditionOperand::URL, WebpageConditionOperand::PAGE_TITLE, and WebpageConditionOperand::PAGE_CONTENT operands.
