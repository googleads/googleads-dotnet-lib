string Google.Api.Ads.AdWords.v201802.CampaignGroup.name
getset

Name of this campaign group. This field is required and should not be

null for ADD operations.

For SET and REMOVE operations, this can be used to address the campaign group by name when the campaign group is ACTIVE or PAUSED. Removed campaign groups cannot be addressed by name. If you wish to rename a campaign group, you must provide the ID. This field can be selected using the value "Name".This field can be filtered on. Campaign group names must not contain any null (code point 0x0), NL line feed (code point 0xA) or carriage return (code point 0xD) characters. This is checked by the regular expression '[^]*'. This field is required and should not be null when it is contained within Operators : ADD. This string must not be empty, (trimmed).
